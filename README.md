# Mani-WorldStream

This is a streaming website project that I built using PHP, MySQL, HTML, CSS and JavaScript.

The main purpose of this project was to learn full-stack web development concepts such as authentication, session handling, database connectivity and responsive UI design.

## Features

- User Registration and Login
- Session Based Authentication
- Home Page
- TV Streaming Page
- Movies Page
- Search Page
- User Profile Page
- Responsive Sidebar Navigation

## Technologies Used

- PHP

- HTML
- CSS
- JavaScript
